borders

To add a border to an element, do the css property border:5px[width of border], solid[type,explained later], black[color];

this image is another example of a border (zoom in the image to see the element with a border).


fun fact, when applying a backround statement to a button, the border automatically changes.


Types of borders:

dotted - Defines a dotted border.

dashed - Defines a dashed border.

solid - Defines a solid border.

double - Defines a double border.

groove - Defines a 3D grooved border. The effect depends on the border-color value.

ridge - Defines a 3D ridged border. The effect depends on the border-color value.

inset - Defines a 3D inset border. The effect depends on the border-color value.

outset - Defines a 3D outset border. The effect depends on the border-color value.

none - Defines no border.

hidden - Defines a hidden border.

mixed - example 3px solid dashed dotted double black;

algining*

Use padding:(how much you want the element to be squezeed in both vertical and horizontal terms),you can also use margin: (it is similar to padding but the margin is the first outer layer and then the padding is the middle layer and the content is the core and main layer. The border is the boundry of the element's actual space (consider the margin the free space reserved for the spacing of an element).



You can also use position:respective;[considers the location of other elements and does not overlap or underlap them],
position:absolute; [never changes window coordinates when the window is resized or you scroll down and does not consider other elements],
and position:fixed;[changes on window resizes and scroll, does not consider other elements].

This video will tell you the difference between position: absolute, fixed, respective


You can use left: distance from left side (eg. 1000px will make it almost right on a laptop), right: distance from right side, top: distance from the top and bottom: distance from the bottom.
